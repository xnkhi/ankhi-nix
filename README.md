# puter
This is the NixOS flake for my own personal Linux system.

This can be used as a reference or guide for your own NixOS flake. Everything is fairly organized and easy to modify.

# wtf is a NixOS?
NixOS is a meta Linux distribution that provides a framework for users to easily create their own Linux systems. You build your own operating system using a functional programming language called Nix. It's very cool, please see [their website](https://nixos.org).
